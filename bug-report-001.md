# Баг-репорт №1

| Поле | Значение |
|---|---|
| **Заголовок** | Не работает авторизация через Apple на сайте www.shop.ru |
| **Описание** | При попытке авторизации на сайте www.shop.ru через Apple-аккаунт выдаётся ошибка |
| **Версия сайта** | v1.92.0 |
| **Окружение** | Яндекс Браузер 26.3.5.782 (64-bit) |
| **Тестовое устройство** | Windows 10 Pro, версия 22H2 |
| **Приоритет** | 🔴 Critical |
| **Автор** | Бурдина Елизавета |
| **Дата** | 15.05.2026 |

---

## Шаги воспроизведения

1. Открыть сайт [www.shop.ru](http://www.shop.ru)
2. Нажать на кнопку «Авторизоваться»
3. Нажать на кнопку со значком Apple для авторизации через Apple-аккаунт

## Фактический результат

Открывается окно с ошибкой *(см. скриншот)*

## Ожидаемый результат

Открывается окно ввода данных Apple-аккаунта
